# Frontend Area

Este diretório contém componentes, hooks e experimentações relacionadas ao desenvolvimento Frontend, utilizando principalmente **React** e **TypeScript**.

## Tópicos Abordados
- React Hooks (useEffect, useMemo, etc.)
- Gerenciamento de Estado
- Componentização e Reusabilidade
- Acessibilidade (A11y)
- CSS-in-JS e Tailwind
